# Ping Pong Game 🏓

A 2-player Ping Pong game built using **Python's Turtle module**.  
Players can control paddles using the keyboard, and the game keeps track of scores for both players.

---

## Features

- Two-player mode
- Player controls:
  - **Player 1:** W (up), S (down)
  - **Player 2:** Up Arrow (up), Down Arrow (down)
- Ball bounces realistically off paddles and walls
- Score tracking for both players
- Simple and engaging graphical interface using Python Turtle
